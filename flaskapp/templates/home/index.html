Welcome to flask-gulp-minimal-boilerplate

This is a starter/boilerplate Flask project.

It includes uikit, Flask-SQLAlchemy, WTForms, Gulp, BrowserSync and various testing utilities out of the box.

Gulp is used to handle the frontend workflow. Included tasks are javascript minification and concatination, SASS compilation and image minification.

BrowserSync automatically reloads web pages when source code changes are saved.
